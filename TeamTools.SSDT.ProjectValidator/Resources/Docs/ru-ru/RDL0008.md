
# Недопустимый символ в имени источника данных

|||
|:-|:-|
| Id | **RDL0008**
| Мнемо | DATASOURCE_NAME_ALLOWED_SYMBOLS
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [DatasourceReferenceNameAlphanumericRule.cs](../../../Rules/Reports/DatasourceReferenceNameAlphanumericRule.cs)

## Причина

Правило срабатывает, если в отчете обнаружено имя источника данных, содержащее недопустимые символы.
<p id="descr">Удалите недопустимые символы из имени источника данных.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="$Ё">
    </DataSource>
  </DataSources>
</Report>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="UNUSED">
    </DataSource>
  </DataSources>
</Report>
```

## Подсказки

💡 Запрещено использование любых символов, кроме букв латинского алфавита, символов тире (-), подчеркивания (_) или точки (.).
