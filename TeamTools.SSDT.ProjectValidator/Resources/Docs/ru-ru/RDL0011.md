
# Ссылка на неизвестный источник данных

|||
|:-|:-|
| Id | **RDL0011**
| Мнемо | UNRESOLVED_DATASOURCE_REFERENCE
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [UnresolvedDatasourceReferenceRule.cs](../../../Rules/Reports/UnresolvedDatasourceReferenceRule.cs)

## Причина

Правило срабатывает, если в отчете есть ссылка на неизвестный источник данных.
<p id="descr">Определите источник данных или используйте один из имеющихся.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="TEST_DS">
    </DataSource>
  </DataSources>
  <DataSets>
    <DataSet Name="MY_DS">
      <Query>
        <DataSourceName>UNKNOWN_DS</DataSourceName>
        <CommandText>...</CommandText>
      </Query>
    </DataSet>
  </DataSets>
</Report>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="TEST_DS">
    </DataSource>
  </DataSources>
  <DataSets>
    <DataSet Name="MY_DS">
      <Query>
        <DataSourceName>TEST_DS</DataSourceName>
        <CommandText>...</CommandText>
      </Query>
    </DataSet>
  </DataSets>
</Report>
```
