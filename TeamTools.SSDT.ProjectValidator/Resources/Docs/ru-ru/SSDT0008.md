
# Задана некорректная область действия курсоров

|||
|:-|:-|
| Id | **SSDT0008**
| Мнемо | SQL_PROJECT_CURSOR_OPTIONS
| Серьёзность | ⚠ Предупреждение
| Категория | [Проект](./Группа_Проект.md)
| Исходный код | [DefaultCursorOptionsRule.cs](../../../Rules/SqlProject/DefaultCursorOptionsRule.cs)

## Причина

Правило срабатывает, если в файле проекта у опции DefaultCursor задано значение, отличное от LOCAL.
<p id="descr">Задайте опции DefaultCursor значение LOCAL.</p>

## Примеры

Некорректно

```xml
<DefaultCursor>GLOBAL</DefaultCursor>
```

Корректно

```xml
<DefaultCursor>LOCAL</DefaultCursor>
```
