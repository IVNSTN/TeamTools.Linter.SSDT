
# Автоматически сгенерированное имя источника данных

|||
|:-|:-|
| Id | **RDL0009**
| Мнемо | DATASOURCE_NAME_NODEFAULT
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [DatasourceReferenceNameAutoGeneratedRule.cs](../../../Rules/Reports/DatasourceReferenceNameAutoGeneratedRule.cs)

## Причина

Правило срабатывает при обнаружении источника данных с автоматически сгенерированным именем.
<p id="descr">Подберите источнику данных более информативное имя.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="Datasource1">
    </DataSource>
  </DataSources>
</Report>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="MY_DS">
    </DataSource>
  </DataSources>
</Report>
```

## Подсказки

💡 К автоматически сгенерированным именам относятся имена вида:

- DatasourceXXX
- Datasource XXX
- Datasource-XXX

где XXX - цифра порядкового номера.
