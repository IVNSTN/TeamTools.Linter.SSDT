
# Имя сервера не полностью определено

|||
|:-|:-|
| Id | **RDS0001**
| Мнемо | SERVER_LINK_FULLY_QUALIFIED
| Серьёзность | ⚠ Предупреждение
| Категория | [ИсточникДанных](./Группа_ИсточникДанных.md)
| Исходный код | [ServerNameFullyQualifiedRule.cs](../../../Rules/ReportDatasource/ServerNameFullyQualifiedRule.cs)

## Причина

Правило срабатывает, если в определении источника данных не указано полное доменное имя (FQDN) сервера, где располагается база данных.
<p id="descr">Укажите полное доменное имя сервера.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv;Initial Catalog=myDb;Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```

## Подсказки

💡 FQDN (Fully Qualified Domain Name) или полное доменное имя сервера представляет собой имя следующего вида: server-name.domain-name.zone. Например: my_srv.my-domain.local
