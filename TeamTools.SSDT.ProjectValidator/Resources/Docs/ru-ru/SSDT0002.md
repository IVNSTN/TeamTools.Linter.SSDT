
# Файл не включен в проект

|||
|:-|:-|
| Id | **SSDT0002**
| Мнемо | PROJECT_FOLDER_DANGLING_FILE
| Серьёзность | ⚠ Предупреждение
| Категория | [Проект](./Группа_Проект.md)
| Исходный код | [DanglingFileInProjectFolderRule.cs](../../../Rules/SqlProject/DanglingFileInProjectFolderRule.cs)

## Причина

Правило срабатывает, если в дереве папок проекта обнаружен не включенный в проект файл.
<p id="descr">Включите файл в проект.</p>

## Примеры

Некорректно

```xml
<Build Include="Stored Procedures\dbo.proc_a.sql" />
<Build Include="Stored Procedures\dbo.proc_b.sql" />
```

Корректно

```xml
<Build Include="Stored Procedures\dbo.proc_a.sql" />
<Build Include="Stored Procedures\dbo.proc_b.sql" />
<Build Include="Stored Procedures\dbo.proc_c.sql" />
```
