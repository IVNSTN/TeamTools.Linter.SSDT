
# Ссылка на неизвестный набор данных

|||
|:-|:-|
| Id | **RDL0013**
| Мнемо | UNRESOLVED_DATASET_REFERENCE
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [UnresolvedDatasetReferenceRule.cs](../../../Rules/Reports/UnresolvedDatasetReferenceRule.cs)

## Причина

Правило срабатывает, если в отчете есть ссылка на неизвестный набор данных.
<p id="descr">Определите набор данных или используйте один из имеющихся.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS">
    </DataSet>
  </DataSets>
  <ReportSections>
    <ReportSection>
      <Body>
        <ReportItems>
          <Tablix Name="Tablix1">
            <DataSetName>UNKNOWN_DS</DataSetName>
          </Tablix>
        </ReportItems>
      </Body>
    </ReportSection>
  </ReportSections>
</Report>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS">
    </DataSet>
  </DataSets>
  <ReportSections>
    <ReportSection>
      <Body>
        <ReportItems>
          <Tablix Name="Tablix1">
            <DataSetName>MY_DS</DataSetName>
          </Tablix>
        </ReportItems>
      </Body>
    </ReportSection>
  </ReportSections>
</Report>
```
