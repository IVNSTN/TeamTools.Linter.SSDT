
# Определение строки подключения в отчёте

|||
|:-|:-|
| Id | **RDL0003**
| Мнемо | CONNECTION_STRING_ILLEGAL
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [ConnectionStringInReportRule.cs](../../../Rules/Reports/ConnectionStringInReportRule.cs)

## Причина

Правило срабатывает, если в отчете обнаружен тег, определяющий строку подключения.
<p id="descr">Удалите строку подключения из отчета.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="TEST_DS">
      <ConnectionProperties>
        <DataProvider>SQL</DataProvider>
        <ConnectString>Data Source=srv.domain.org;Initial Catalog=reporting_database</ConnectString>
        <IntegratedSecurity>true</IntegratedSecurity>
      </ConnectionProperties>
    </DataSource>
  </DataSources>
</Report>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="TEST_DS">
      <ConnectionProperties>
        <DataProvider>SQL</DataProvider>
      </ConnectionProperties>
    </DataSource>
  </DataSources>
</Report>
```
