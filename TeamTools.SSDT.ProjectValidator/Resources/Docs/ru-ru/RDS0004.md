
# Слишком короткое имя источника данных

|||
|:-|:-|
| Id | **RDS0004**
| Мнемо | DATASOURCE_NAME_LENGTH
| Серьёзность | ⚠ Предупреждение
| Категория | [ИсточникДанных](./Группа_ИсточникДанных.md)
| Исходный код | [DatasourceNameMinLengthRule.cs](../../../Rules/ReportDatasource/DatasourceNameMinLengthRule.cs)

## Причина

Правило срабатывает, если обнаружено имя источника данных длиной в 1 символ.
<p id="descr">Подберите источнику данных более информативное имя.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="A">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```
