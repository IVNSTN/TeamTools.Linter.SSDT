
# Неиспользуемый источник данных

|||
|:-|:-|
| Id | **RDL0010**
| Мнемо | UNUSED_DATASOURCE
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [DatasourceNeverUserRule.cs](../../../Rules/Reports/DatasourceNeverUserRule.cs)

## Причина

Правило срабатывает, если в отчете есть объявленный, но не используемый источник данных.
<p id="descr">Удалите неиспользуемый источник данных.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="TEST_DS"/>
    <DataSource Name="UNUSED"/><!-- here -->
  </DataSources>
  <DataSets>
    <DataSet Name="MY_DS">
      <Query>
        <DataSourceName>TEST_DS</DataSourceName>
        <CommandText>...</CommandText>
      </Query>
    </DataSet>
  </DataSets>
</Report>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="TEST_DS"/>
  </DataSources>
  <DataSets>
    <DataSet Name="MY_DS">
      <Query>
        <DataSourceName>TEST_DS</DataSourceName>
        <CommandText>...</CommandText>
      </Query>
    </DataSet>
  </DataSets>
</Report>
```
