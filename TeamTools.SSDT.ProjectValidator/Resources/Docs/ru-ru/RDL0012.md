
# Неиспользуемый набор данных

|||
|:-|:-|
| Id | **RDL0012**
| Мнемо | UNUSED_DATASET
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [DatasetNeverUsedRule.cs](../../../Rules/Reports/DatasetNeverUsedRule.cs)

## Причина

Правило срабатывает, если в отчете есть объявленный, но не используемый набор данных.
<p id="descr">Удалите неиспользуемый набор данных.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS"/>
    <DataSet Name="UNUSED"/> <!-- here -->
  </DataSets>
  <ReportSections>
    <ReportSection>
      <Body>
        <ReportItems>
          <Tablix Name="Tablix1">
            <DataSetName>MY_DS</DataSetName>
          </Tablix>
        </ReportItems>
      </Body>
    </ReportSection>
  </ReportSections>
</Report>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS">
    </DataSet>
  </DataSets>
  <ReportSections>
    <ReportSection>
      <Body>
        <ReportItems>
          <Tablix Name="Tablix1">
            <DataSetName>MY_DS</DataSetName>
          </Tablix>
        </ReportItems>
      </Body>
    </ReportSection>
  </ReportSections>
</Report>
```
