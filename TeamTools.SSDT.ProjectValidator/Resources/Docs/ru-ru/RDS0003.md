
# Не задана строка подключения

|||
|:-|:-|
| Id | **RDS0003**
| Мнемо | CONNECTION_STRING_UNDEFINED
| Серьёзность | ⚠ Предупреждение
| Категория | [ИсточникДанных](./Группа_ИсточникДанных.md)
| Исходный код | [ConnectionStringDefinedRule.cs](../../../Rules/ReportDatasource/ConnectionStringDefinedRule.cs)

## Причина

Правило срабатывает, если в источнике данных не определена строка подключения.
<p id="descr">Определите строку подключения.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
  </ConnectionProperties>
</RptDataSource>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```
