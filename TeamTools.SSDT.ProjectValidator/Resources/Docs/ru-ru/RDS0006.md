
# Автоматически сгенерированное имя источника данных

|||
|:-|:-|
| Id | **RDS0006**
| Мнемо | DATASOURCE_NAME_NODEFAULT
| Серьёзность | ⚠ Предупреждение
| Категория | [ИсточникДанных](./Группа_ИсточникДанных.md)
| Исходный код | [DatasourceNameAutoGeneratedRule.cs](../../../Rules/ReportDatasource/DatasourceNameAutoGeneratedRule.cs)

## Причина

Правило срабатывает при обнаружении источника данных с автоматически сгенерированным именем.
<p id="descr">Подберите источнику данных более информативное имя.</p>

## Примеры

Некорректно

```xml
<RptDataSource Name="Datasource1">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```

## Подсказки

💡 К автоматически сгенерированным именам относятся имена вида:

- DatasourceXXX
- Datasource XXX
- Datasource-XXX

где XXX - цифра порядкового номера.
