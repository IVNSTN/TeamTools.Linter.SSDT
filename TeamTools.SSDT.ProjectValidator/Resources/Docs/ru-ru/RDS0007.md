
# Имя источника данных не соответствует шаблону

|||
|:-|:-|
| Id | **RDS0007**
| Мнемо | DATASOURCE_NAME_TEMPLATE
| Серьёзность | ⚠ Предупреждение
| Категория | [ИсточникДанных](./Группа_ИсточникДанных.md)
| Исходный код | [DatasourceNameTemplateRule.cs](../../../Rules/ReportDatasource/DatasourceNameTemplateRule.cs)

## Причина

Правило срабатывает, если имя источника данных не соответствует шаблону \<srv\>.\<db\>
<p id="descr">Приведите имя источника данных в соответствие шаблону.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```
