
# Недопустимое значение опции IntegratedSecurity

|||
|:-|:-|
| Id | **RDS0002**
| Мнемо | INTEGRATED_SECURITY_VALUE
| Серьёзность | ⚠ Предупреждение
| Категория | [ИсточникДанных](./Группа_ИсточникДанных.md)
| Исходный код | [IntegratedSecurityInDatasourceRule.cs](../../../Rules/ReportDatasource/IntegratedSecurityInDatasourceRule.cs)

## Причина

Правило срабатывает, если для опции IntegratedSecurity задано значение, отличное от SSPI.
<p id="descr">Используйте IntegratedSecurity=SSPI.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=wrong</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```
