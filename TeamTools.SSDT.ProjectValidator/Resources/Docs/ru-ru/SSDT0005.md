
# Свойство CopyLocal/Private не требует явного указания True

|||
|:-|:-|
| Id | **SSDT0005**
| Мнемо | DACPAC_COPY_LOCAL
| Серьёзность | ⚠ Предупреждение
| Категория | [Проект](./Группа_Проект.md)
| Исходный код | [DacpacCopyLocalRule.cs](../../../Rules/SqlProject/DacpacCopyLocalRule.cs)

## Причина

Правило срабатывает, если свойство CopyLocal/Private подключенного dacpac-файлом проекта выставлено в True в явном виде.
<p id="descr">Удалите явное задание True для CopyLocal/Private</p>

## Примеры

Некорректно

```xml
<ArtifactReference Include="..\..\_DACPACs\another_project.dacpac">
  <HintPath>..\..\_DACPACs\another_project.dacpac</HintPath>
  <SuppressMissingDependenciesErrors>True</SuppressMissingDependenciesErrors>
  <DatabaseSqlCmdVariable>another_project</DatabaseSqlCmdVariable>
  <Private>True</Private>
</ArtifactReference>
```

Корректно

```xml
<ArtifactReference Include="..\..\_DACPACs\another_project.dacpac">
  <HintPath>..\..\_DACPACs\another_project.dacpac</HintPath>
  <SuppressMissingDependenciesErrors>True</SuppressMissingDependenciesErrors>
  <DatabaseSqlCmdVariable>another_project</DatabaseSqlCmdVariable>
</ArtifactReference>
```
