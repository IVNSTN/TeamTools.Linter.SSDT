
# Некорректное значение опции IntegratedSecurity

|||
|:-|:-|
| Id | **RDL0002**
| Мнемо | INTEGRATED_SECURITY_OFF
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [IntegratedSecurityInReportRule.cs](../../../Rules/Reports/IntegratedSecurityInReportRule.cs)

## Причина

Правило срабатывает, если для опции IntegratedSecurity выставлено значение, отличное от None.
<p id="descr">Удалите опцию IntegratedSecurity или используйте значение None.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="MY_DS">
      <ConnectionProperties>
        <IntegratedSecurity>SSPI</IntegratedSecurity>
      </ConnectionProperties>
    </DataSource>
  </DataSources>
</Report>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="MY_DS">
      <ConnectionProperties>
      </ConnectionProperties>
    </DataSource>
  </DataSources>
</Report>
```
