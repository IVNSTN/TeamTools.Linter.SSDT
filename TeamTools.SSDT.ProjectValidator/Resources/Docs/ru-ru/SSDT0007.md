
# Использование устаревших опций проекта

|||
|:-|:-|
| Id | **SSDT0007**
| Мнемо | SQL_PROJECT_OBSOLETE_OPTIONS
| Серьёзность | ⚠ Предупреждение
| Категория | [Проект](./Группа_Проект.md)
| Исходный код | [SqlProjObsoleteOptionsRule.cs](../../../Rules/SqlProject/SqlProjObsoleteOptionsRule.cs)

## Причина

Правило срабатывает, если в файле проекта заданы значения для опций, унаследованных от устаревших типов проектов.
<p id="descr">Удалите устаревшие опции.</p>

## Примеры

Некорректно

```xml
<DefaultFileStructure>BySchemaType</DefaultFileStructure>
<IncludeSchemaNameInFileName>True</IncludeSchemaNameInFileName>
<OutputType>Database</OutputType>
<DeployToDatabase>True</DeployToDatabase>
<SccProjectName></SccProjectName>
<SccProvider></SccProvider>
```

Корректно

```xml
<DefaultFileStructure>BySchemaType</DefaultFileStructure>
<IncludeSchemaNameInFileName>True</IncludeSchemaNameInFileName>
<OutputType>Database</OutputType>
<DeployToDatabase>True</DeployToDatabase>
```

## Подсказки

💡 К опциям устаревших проектов относятся:

- SccProjectName
- SccProvider
- SccAuxPath
- SccLocalPath
