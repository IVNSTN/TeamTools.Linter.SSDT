
# Некорректно заданы опции, определяющие структуру проекта

|||
|:-|:-|
| Id | **SSDT0006**
| Мнемо | SQL_PROJECT_STRUCT_OPTIONS
| Серьёзность | ⚠ Предупреждение
| Категория | [Проект](./Группа_Проект.md)
| Исходный код | [SqlProjStructureOptionsRule.cs](../../../Rules/SqlProject/SqlProjStructureOptionsRule.cs)

## Причина

Правило срабатывает, если заданы некорректные или дублирующиеся значения опций, определяющих структуру проекта.
<p id="descr">Задайте корректные значения опций структуры проекта.</p>

## Примеры

Некорректно

```xml
<DefaultFileStructure>BySchemaType</DefaultFileStructure>
<DefaultFileStructure>BySchemaType</DefaultFileStructure>
<IncludeSchemaNameInFileName>False</IncludeSchemaNameInFileName>
<!-- missing is not ok OutputType -->
<!-- missing is not ok DeployToDatabase -->
```

Корректно

```xml
<DefaultFileStructure>BySchemaType</DefaultFileStructure>
<IncludeSchemaNameInFileName>True</IncludeSchemaNameInFileName>
<OutputType>Database</OutputType>
<DeployToDatabase>True</DeployToDatabase>
```

## Подсказки

💡 Правило проверяет наличие и значения следующих опций:

- DefaultFileStructure = BySchemaType
- IncludeSchemaNameInFileName = True
- OutputType = Database
- DeployToDatabase = True
