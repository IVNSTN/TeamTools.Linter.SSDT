
# Переменная из файла проекта не добавлена в publish.xml

|||
|:-|:-|
| Id | **SSDT0010**
| Мнемо | PUBLISH_XML_VARIABLE_MISSING
| Серьёзность | ⚠ Предупреждение
| Категория | [Проект](./Группа_Проект.md)
| Исходный код | [PublishXmlHasSameValuesListRule.cs](../../../Rules/SqlProject/PublishXmlHasSameValuesListRule.cs)

## Причина

Правило срабатывает, если обнаружена переменная, объявленная в файле проекта, которая не определена в файле конфигурации для публикации (publish.xml).
<p id="descr">Добавьте переменную в publish.xml.</p>

## Примеры

Некорректно

```xml
<!-- sqlproj fragment -->
  <ArtifactReference Include="..\..\_DACPACs\another_project.dacpac"><!-- ... -->
  <ArtifactReference Include="..\..\_DACPACs\my_project.dacpac"><!-- ... -->
<!-- ... -->
  <SqlCmdVariable Include="another_project"><!-- ... -->
  <SqlCmdVariable Include="my_project"><!-- ... -->

<!-- publish.xml fragment -->
<ItemGroup>
    <SqlCmdVariable Include="my_project"><!-- ... -->
</ItemGroup>
```

Корректно

```xml
<!-- sqlproj fragment -->
  <ArtifactReference Include="..\..\_DACPACs\another_project.dacpac"><!-- ... -->
  <ArtifactReference Include="..\..\_DACPACs\my_project.dacpac"><!-- ... -->
<!-- ... -->
  <SqlCmdVariable Include="another_project"><!-- ... -->
  <SqlCmdVariable Include="my_project"><!-- ... -->

<!-- publish.xml fragment -->
<ItemGroup>
  <SqlCmdVariable Include="another_project"><!-- ... -->
  <SqlCmdVariable Include="my_project"><!-- ... -->
</ItemGroup>
```
