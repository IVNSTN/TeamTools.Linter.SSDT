
# Слишком короткое имя источника данных

|||
|:-|:-|
| Id | **RDL0007**
| Мнемо | DATASOURCE_NAME_LENGTH
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [DatasourceReferenceNameMinLengthRule.cs](../../../Rules/Reports/DatasourceReferenceNameMinLengthRule.cs)

## Причина

Правило срабатывает, если в отчете обнаружено имя источника данных длиной в 1 символ.
<p id="descr">Подберите источнику данных более информативное имя.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="..">
  <DataSources>
    <DataSource Name="D">
    </DataSource>
  </DataSources>
</Report>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="MY_DS">
    </DataSource>
  </DataSources>
</Report>
```
