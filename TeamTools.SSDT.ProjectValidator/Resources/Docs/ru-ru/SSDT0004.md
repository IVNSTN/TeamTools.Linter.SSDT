
# Неправильный путь в HintPath

|||
|:-|:-|
| Id | **SSDT0004**
| Мнемо | DACPAC_HINT_PATH
| Серьёзность | ⚠ Предупреждение
| Категория | [Проект](./Группа_Проект.md)
| Исходный код | [DacpacHintPathEqualsIncludePathRule.cs](../../../Rules/SqlProject/DacpacHintPathEqualsIncludePathRule.cs)

## Причина

Правило срабатывает, если у подключенного dacpac-файлом проекта путь, указанный в свойстве HintPath, не совпадает с путем, заданным в атрибуте Include.
<p id="descr">Задайте правильный путь в HintPath.</p>

## Примеры

Некорректно

```xml
<ArtifactReference Include="..\..\_DACPACs\another_project.dacpac">
  <HintPath>C:\Projects\_DACPACs\another_project.dacpac</HintPath>
  <SuppressMissingDependenciesErrors>True</SuppressMissingDependenciesErrors>
  <DatabaseSqlCmdVariable>another_project</DatabaseSqlCmdVariable>
</ArtifactReference>
```

Корректно

```xml
<ArtifactReference Include="..\..\_DACPACs\another_project.dacpac">
  <HintPath>..\..\_DACPACs\another_project.dacpac</HintPath>
  <SuppressMissingDependenciesErrors>True</SuppressMissingDependenciesErrors>
  <DatabaseSqlCmdVariable>another_project</DatabaseSqlCmdVariable>
</ArtifactReference>
```
