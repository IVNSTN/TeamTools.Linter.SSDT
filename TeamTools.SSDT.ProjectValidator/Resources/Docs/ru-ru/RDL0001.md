
# Встраивание изображений в отчет

|||
|:-|:-|
| Id | **RDL0001**
| Мнемо | EMBEDDED_IMAGES
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [EmbeddedImagesRule.cs](../../../Rules/Reports/EmbeddedImagesRule.cs)

## Причина

Правило срабатывает, если в отчете найдено встроенное, при помощи тега EmbeddedImages, изображение.
<p id="descr">Удалите встроенное в отчет изображение.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <EmbeddedImages></EmbeddedImages>
</Report>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
</Report>
```
