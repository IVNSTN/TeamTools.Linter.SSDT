
# Недопустимый символ в имени набора данных

|||
|:-|:-|
| Id | **RDL0005**
| Мнемо | DATASET_NAME_ALLOWED_SYMBOLS
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [DatasetNameAlphanumericRule.cs](../../../Rules/Reports/DatasetNameAlphanumericRule.cs)

## Причина

Правило срабатывает, если в отчете обнаружено имя набора данных, содержащее недопустимые символы.
<p id="descr">Удалите недопустимые символы из имени набора данных.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="$Ё">
    </DataSet>
  </DataSets>
</Report>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS">
    </DataSet>
  </DataSets>
</Report>
```

## Подсказки

💡 Запрещено использование любых символов, кроме букв латинского алфавита, символов тире (-), подчеркивания (_) или точки (.).
