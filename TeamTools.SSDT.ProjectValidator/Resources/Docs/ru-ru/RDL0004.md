
# Слишком короткое имя набора данных

|||
|:-|:-|
| Id | **RDL0004**
| Мнемо | DATASET_NAME_LENGTH
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [DatasetNameMinLengthRule.cs](../../../Rules/Reports/DatasetNameMinLengthRule.cs)

## Причина

Правило срабатывает, если в отчете обнаружено имя набора данных длиной в 1 символ.
<p id="descr">Подберите набору данных более информативное имя.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="D">
    </DataSet>
  </DataSets>
</Report>
```

Корректно

```xml
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS">
    </DataSet>
  </DataSets>
</Report>
```
