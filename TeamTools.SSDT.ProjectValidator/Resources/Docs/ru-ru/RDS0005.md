
# Недопустимый символ в имени источника данных

|||
|:-|:-|
| Id | **RDS0005**
| Мнемо | DATASOURCE_NAME_ALLOWED_SYMBOLS
| Серьёзность | ⚠ Предупреждение
| Категория | [ИсточникДанных](./Группа_ИсточникДанных.md)
| Исходный код | [DatasourceNameAlphanumericRule.cs](../../../Rules/ReportDatasource/DatasourceNameAlphanumericRule.cs)

## Причина

Правило срабатывает, если имя источника данных содержит недопустимые символы.
<p id="descr">Удалите недопустимые символы из имени источника данных.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="$Ё">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```

## Подсказки

💡 Запрещено использование любых символов, кроме букв латинского алфавита, символов тире (-), подчеркивания (_) или точки (.).
