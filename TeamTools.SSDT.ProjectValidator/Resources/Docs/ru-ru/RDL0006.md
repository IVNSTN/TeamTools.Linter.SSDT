
# Автоматически сгенерированное имя набора данных

|||
|:-|:-|
| Id | **RDL0006**
| Мнемо | DATASET_NAME_NODEFAULT
| Серьёзность | ⚠ Предупреждение
| Категория | [Отчет](./Группа_Отчет.md)
| Исходный код | [DatasetNameAutoGeneratedRule.cs](../../../Rules/Reports/DatasetNameAutoGeneratedRule.cs)

## Причина

Правило срабатывает при обнаружении набора данных с автоматически сгенерированным именем.
<p id="descr">Подберите набору данных более информативное имя.</p>

## Примеры

Некорректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="Dataset1">
    </DataSet>
  </DataSets>
</Report>
```

Корректно

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS">
    </DataSet>
  </DataSets>
</Report>
```

## Подсказки

💡 К автоматически сгенерированным именам относятся имена вида:

- DatasetXXX
- Dataset XXX
- Dataset-XXX

где XXX - цифра порядкового номера.
