
# Unresolved DataSource reference

|||
|:-|:-|
| Id | **RDL0011**
| Mnemo | UNRESOLVED_DATASOURCE_REFERENCE
| Severity | ⚠ Warning
| Category | [Report](./Group_Report.md)
| Source code | [UnresolvedDatasourceReferenceRule.cs](../../../Rules/Reports/UnresolvedDatasourceReferenceRule.cs)

## Cause

The rule is triggered if a report contains an unresolved DataSource reference.
<p id="descr">Define the DataSource or use the existing one.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="TEST_DS">
    </DataSource>
  </DataSources>
  <DataSets>
    <DataSet Name="MY_DS">
      <Query>
        <DataSourceName>UNKNOWN_DS</DataSourceName>
        <CommandText>...</CommandText>
      </Query>
    </DataSet>
  </DataSets>
</Report>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="TEST_DS">
    </DataSource>
  </DataSources>
  <DataSets>
    <DataSet Name="MY_DS">
      <Query>
        <DataSourceName>TEST_DS</DataSourceName>
        <CommandText>...</CommandText>
      </Query>
    </DataSet>
  </DataSets>
</Report>
```
