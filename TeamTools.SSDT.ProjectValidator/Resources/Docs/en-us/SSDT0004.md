
# Dacpac reference HintPath must equal include path

|||
|:-|:-|
| Id | **SSDT0004**
| Mnemo | DACPAC_HINT_PATH
| Severity | ⚠ Warning
| Category | [Project](./Group_Project.md)
| Source code | [DacpacHintPathEqualsIncludePathRule.cs](../../../Rules/SqlProject/DacpacHintPathEqualsIncludePathRule.cs)

## Cause

This rule is triggered if the path specified in the HintPath property of the referenced dacpac does not match the path specified in the Include attribute.
<p id="descr">Specify the correct path in HintPath.</p>

## Examples

Bad

```xml
<ArtifactReference Include="..\..\_DACPACs\another_project.dacpac">
  <HintPath>C:\Projects\_DACPACs\another_project.dacpac</HintPath>
  <SuppressMissingDependenciesErrors>True</SuppressMissingDependenciesErrors>
  <DatabaseSqlCmdVariable>another_project</DatabaseSqlCmdVariable>
</ArtifactReference>
```

Good

```xml
<ArtifactReference Include="..\..\_DACPACs\another_project.dacpac">
  <HintPath>..\..\_DACPACs\another_project.dacpac</HintPath>
  <SuppressMissingDependenciesErrors>True</SuppressMissingDependenciesErrors>
  <DatabaseSqlCmdVariable>another_project</DatabaseSqlCmdVariable>
</ArtifactReference>
```
