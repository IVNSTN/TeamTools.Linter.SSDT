
# ConnectionString is missing

|||
|:-|:-|
| Id | **RDS0003**
| Mnemo | CONNECTION_STRING_UNDEFINED
| Severity | ⚠ Warning
| Category | [Datasource](./Group_Datasource.md)
| Source code | [ConnectionStringDefinedRule.cs](../../../Rules/ReportDatasource/ConnectionStringDefinedRule.cs)

## Cause

This rule is triggered if the data source does not have a ConnectionString defined.
<p id="descr">Define a ConnectionString.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
  </ConnectionProperties>
</RptDataSource>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```
