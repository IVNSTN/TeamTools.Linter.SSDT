
# Incorrect value for the IntegratedSecurity option

|||
|:-|:-|
| Id | **RDL0002**
| Mnemo | INTEGRATED_SECURITY_OFF
| Severity | ⚠ Warning
| Category | [Report](./Group_Report.md)
| Source code | [IntegratedSecurityInReportRule.cs](../../../Rules/Reports/IntegratedSecurityInReportRule.cs)

## Cause

This rule is triggered if the IntegratedSecurity option is set to a value other than None.
<p id="descr">Remove the IntegratedSecurity option, or use the None value.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="MY_DS">
      <ConnectionProperties>
        <IntegratedSecurity>SSPI</IntegratedSecurity>
      </ConnectionProperties>
    </DataSource>
  </DataSources>
</Report>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="MY_DS">
      <ConnectionProperties>
      </ConnectionProperties>
    </DataSource>
  </DataSources>
</Report>
```
