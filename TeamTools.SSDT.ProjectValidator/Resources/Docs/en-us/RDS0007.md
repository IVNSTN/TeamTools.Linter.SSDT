
# DataSource name does not match the template

|||
|:-|:-|
| Id | **RDS0007**
| Mnemo | DATASOURCE_NAME_TEMPLATE
| Severity | ⚠ Warning
| Category | [Datasource](./Group_Datasource.md)
| Source code | [DatasourceNameTemplateRule.cs](../../../Rules/ReportDatasource/DatasourceNameTemplateRule.cs)

## Cause

This rule is triggered if the DataSource name does not match the template \<srv>.\<db>
<p id="descr">Make the DataSource name match the template.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```
