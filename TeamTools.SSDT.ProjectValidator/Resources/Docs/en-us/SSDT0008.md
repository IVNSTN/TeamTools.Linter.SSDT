
# Cursor scope option has wrong value

|||
|:-|:-|
| Id | **SSDT0008**
| Mnemo | SQL_PROJECT_CURSOR_OPTIONS
| Severity | ⚠ Warning
| Category | [Project](./Group_Project.md)
| Source code | [DefaultCursorOptionsRule.cs](../../../Rules/SqlProject/DefaultCursorOptionsRule.cs)

## Cause

The rule is triggered if the DefaultCursor option in the project file is set to a value other than LOCAL.
<p id="descr">Set DefaultCursor to LOCAL.</p>

## Examples

Bad

```xml
<DefaultCursor>GLOBAL</DefaultCursor>
```

Good

```xml
<DefaultCursor>LOCAL</DefaultCursor>
```
