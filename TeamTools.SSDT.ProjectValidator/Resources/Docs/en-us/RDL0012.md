
# DataSet defined but never used

|||
|:-|:-|
| Id | **RDL0012**
| Mnemo | UNUSED_DATASET
| Severity | ⚠ Warning
| Category | [Report](./Group_Report.md)
| Source code | [DatasetNeverUsedRule.cs](../../../Rules/Reports/DatasetNeverUsedRule.cs)

## Cause

This rule is triggered if a DataSet was declared but never used.
<p id="descr">Remove the unused DataSet.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS"/>
    <DataSet Name="UNUSED"/> <!-- here -->
  </DataSets>
  <ReportSections>
    <ReportSection>
      <Body>
        <ReportItems>
          <Tablix Name="Tablix1">
            <DataSetName>MY_DS</DataSetName>
          </Tablix>
        </ReportItems>
      </Body>
    </ReportSection>
  </ReportSections>
</Report>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS">
    </DataSet>
  </DataSets>
  <ReportSections>
    <ReportSection>
      <Body>
        <ReportItems>
          <Tablix Name="Tablix1">
            <DataSetName>MY_DS</DataSetName>
          </Tablix>
        </ReportItems>
      </Body>
    </ReportSection>
  </ReportSections>
</Report>
```
