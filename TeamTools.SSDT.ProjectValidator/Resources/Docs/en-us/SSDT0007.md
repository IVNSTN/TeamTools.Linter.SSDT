
# Obsolete project option must be removed

|||
|:-|:-|
| Id | **SSDT0007**
| Mnemo | SQL_PROJECT_OBSOLETE_OPTIONS
| Severity | ⚠ Warning
| Category | [Project](./Group_Project.md)
| Source code | [SqlProjObsoleteOptionsRule.cs](../../../Rules/SqlProject/SqlProjObsoleteOptionsRule.cs)

## Cause

This rule is triggered if obsolete project options are used.
<p id="descr">Remove the obsolete option.</p>

## Examples

Bad

```xml
<DefaultFileStructure>BySchemaType</DefaultFileStructure>
<IncludeSchemaNameInFileName>True</IncludeSchemaNameInFileName>
<OutputType>Database</OutputType>
<DeployToDatabase>True</DeployToDatabase>
<SccProjectName></SccProjectName>
<SccProvider></SccProvider>
```

Good

```xml
<DefaultFileStructure>BySchemaType</DefaultFileStructure>
<IncludeSchemaNameInFileName>True</IncludeSchemaNameInFileName>
<OutputType>Database</OutputType>
<DeployToDatabase>True</DeployToDatabase>
```

## Tips

💡 Obsolete project option list:

- SccProjectName
- SccProvider
- SccAuxPath
- SccLocalPath
