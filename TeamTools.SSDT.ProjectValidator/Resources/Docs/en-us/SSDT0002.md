
# Dangling file

|||
|:-|:-|
| Id | **SSDT0002**
| Mnemo | PROJECT_FOLDER_DANGLING_FILE
| Severity | ⚠ Warning
| Category | [Project](./Group_Project.md)
| Source code | [DanglingFileInProjectFolderRule.cs](../../../Rules/SqlProject/DanglingFileInProjectFolderRule.cs)

## Cause

This rule is triggered if a file that is not included in the project is found in the project folder tree.
<p id="descr">Include the file in the project.</p>

## Examples

Bad

```xml
<Build Include="Stored Procedures\dbo.proc_a.sql" />
<Build Include="Stored Procedures\dbo.proc_b.sql" />
```

Good

```xml
<Build Include="Stored Procedures\dbo.proc_a.sql" />
<Build Include="Stored Procedures\dbo.proc_b.sql" />
<Build Include="Stored Procedures\dbo.proc_c.sql" />
```
