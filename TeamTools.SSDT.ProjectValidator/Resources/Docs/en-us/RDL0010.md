
# DataSource defined but never used

|||
|:-|:-|
| Id | **RDL0010**
| Mnemo | UNUSED_DATASOURCE
| Severity | ⚠ Warning
| Category | [Report](./Group_Report.md)
| Source code | [DatasourceNeverUserRule.cs](../../../Rules/Reports/DatasourceNeverUserRule.cs)

## Cause

This rule is triggered if a report has a defined, but never used, DataSource.
<p id="descr">Remove the unused DataSource.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="TEST_DS"/>
    <DataSource Name="UNUSED"/><!-- here -->
  </DataSources>
  <DataSets>
    <DataSet Name="MY_DS">
      <Query>
        <DataSourceName>TEST_DS</DataSourceName>
        <CommandText>...</CommandText>
      </Query>
    </DataSet>
  </DataSets>
</Report>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="TEST_DS"/>
  </DataSources>
  <DataSets>
    <DataSet Name="MY_DS">
      <Query>
        <DataSourceName>TEST_DS</DataSourceName>
        <CommandText>...</CommandText>
      </Query>
    </DataSet>
  </DataSets>
</Report>
```
