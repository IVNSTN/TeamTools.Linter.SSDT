
# Project struct option has wrong value

|||
|:-|:-|
| Id | **SSDT0006**
| Mnemo | SQL_PROJECT_STRUCT_OPTIONS
| Severity | ⚠ Warning
| Category | [Project](./Group_Project.md)
| Source code | [SqlProjStructureOptionsRule.cs](../../../Rules/SqlProject/SqlProjStructureOptionsRule.cs)

## Cause

This rule is triggered if incorrect or duplicate values ​​are specified for the options that define the project structure.
<p id="descr">Specify correct values ​​for the project structure options.</p>

## Examples

Bad

```xml
<DefaultFileStructure>BySchemaType</DefaultFileStructure>
<DefaultFileStructure>BySchemaType</DefaultFileStructure>
<IncludeSchemaNameInFileName>False</IncludeSchemaNameInFileName>
<!-- missing is not ok OutputType -->
<!-- missing is not ok DeployToDatabase -->
```

Good

```xml
<DefaultFileStructure>BySchemaType</DefaultFileStructure>
<IncludeSchemaNameInFileName>True</IncludeSchemaNameInFileName>
<OutputType>Database</OutputType>
<DeployToDatabase>True</DeployToDatabase>
```

## Tips

💡 The presence and values ​​of the following options are checked:

- DefaultFileStructure = BySchemaType
- IncludeSchemaNameInFileName = True
- OutputType = Database
- DeployToDatabase = True
