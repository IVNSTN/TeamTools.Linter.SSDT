
# DataSet name is too short

|||
|:-|:-|
| Id | **RDL0004**
| Mnemo | DATASET_NAME_LENGTH
| Severity | ⚠ Warning
| Category | [Report](./Group_Report.md)
| Source code | [DatasetNameMinLengthRule.cs](../../../Rules/Reports/DatasetNameMinLengthRule.cs)

## Cause

This rule is triggered if the length of a DataSet's name is 1.
<p id="descr">Give the DataSet a more meaningful name.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="D">
    </DataSet>
  </DataSets>
</Report>
```

Good

```xml
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS">
    </DataSet>
  </DataSets>
</Report>
```
