
# IntegratedSecurity option is incorrect

|||
|:-|:-|
| Id | **RDS0002**
| Mnemo | INTEGRATED_SECURITY_VALUE
| Severity | ⚠ Warning
| Category | [Datasource](./Group_Datasource.md)
| Source code | [IntegratedSecurityInDatasourceRule.cs](../../../Rules/ReportDatasource/IntegratedSecurityInDatasourceRule.cs)

## Cause

This rule is triggered if the IntegratedSecurity option is set to a value other than SSPI.
<p id="descr">Use IntegratedSecurity=SSPI.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=wrong</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<RptDataSource Name="srv.myDb">
  <ConnectionProperties>
    <Extension>SQL</Extension>
    <ConnectString>Data Source=srv.domain.org;Initial Catalog=myDb;Integrated Security=SSPI</ConnectString>
  </ConnectionProperties>
</RptDataSource>
```
