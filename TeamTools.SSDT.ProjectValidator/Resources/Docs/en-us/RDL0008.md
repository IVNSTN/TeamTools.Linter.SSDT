
# DataSource name contains illegal symbols

|||
|:-|:-|
| Id | **RDL0008**
| Mnemo | DATASOURCE_NAME_ALLOWED_SYMBOLS
| Severity | ⚠ Warning
| Category | [Report](./Group_Report.md)
| Source code | [DatasourceReferenceNameAlphanumericRule.cs](../../../Rules/Reports/DatasourceReferenceNameAlphanumericRule.cs)

## Cause

This rule is triggered if a DataSource name contains illegal symbols.
<p id="descr">Remove all illegal symbols from the DataSource name.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="$Ё">
    </DataSource>
  </DataSources>
</Report>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="UNUSED">
    </DataSource>
  </DataSources>
</Report>
```

## Tips

💡 It is prohibited to use any symbols other than Latin letters, dashes (-), underscores (_) or periods (.).
