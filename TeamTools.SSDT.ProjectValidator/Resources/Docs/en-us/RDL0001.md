
# Embedding images into a report

|||
|:-|:-|
| Id | **RDL0001**
| Mnemo | EMBEDDED_IMAGES
| Severity | ⚠ Warning
| Category | [Report](./Group_Report.md)
| Source code | [EmbeddedImagesRule.cs](../../../Rules/Reports/EmbeddedImagesRule.cs)

## Cause

This rule is triggered if the EmbeddedImages tag is found.
<p id="descr">Remove all embedded images, along with the EmbeddedImages tag.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <EmbeddedImages></EmbeddedImages>
</Report>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
</Report>
```
