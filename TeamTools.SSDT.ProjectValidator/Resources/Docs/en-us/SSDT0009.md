
# Absolute path used instead of relative

|||
|:-|:-|
| Id | **SSDT0009**
| Mnemo | PROJECT_INCLUDE_ABSOLUTE_PATH
| Severity | ⚠ Warning
| Category | [Project](./Group_Project.md)
| Source code | [AbsolutePathRule.cs](../../../Rules/SqlProject/AbsolutePathRule.cs)

## Cause

This rule is triggered if absolute paths are used in the project file instead of relative ones.
<p id="descr">Use a relative path.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<Project DefaultTargets="Build" xmlns="http://schemas.microsoft.com/developer/msbuild/2003" ToolsVersion="4.0">
  <Import Project="c:\repo\Microsoft.Common.props" Condition="Exists('c:\repo\Microsoft.Common.props')" /> <!-- here -->
  <!-- ... -->
  <PropertyGroup Condition=" '$(Configuration)|$(Platform)' == 'Release|AnyCPU' ">
    <OutputPath>c:\bin\Release\</OutputPath> <!-- here -->
    <BuildScriptName>c:\build.sql</BuildScriptName> <!-- here -->
    <!-- ... -->
  </PropertyGroup>
  <ItemGroup>
    <Build Include="c:\Tables\MyTable.sql" /> <!-- here -->
    <None Include="\\drive\Publish\DB.publish.xml" /> <!-- here -->
  </ItemGroup>
  <ItemGroup>
    <ArtifactReference Include="C:\_DACPACs\frontend.dacpac"><!-- here -->
      <!-- ... -->
    </ArtifactReference>
  </ItemGroup>
</Project>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<Project DefaultTargets="Build" xmlns="http://schemas.microsoft.com/developer/msbuild/2003" ToolsVersion="4.0">
  <Import Project="$(MSBuildExtensionsPath)\$(MSBuildToolsVersion)\Microsoft.Common.props" Condition="Exists('$(MSBuildExtensionsPath)\$(MSBuildToolsVersion)\Microsoft.Common.props')" /> <!-- ... -->
  <PropertyGroup Condition=" '$(Configuration)|$(Platform)' == 'Release|AnyCPU' ">
    <OutputPath>.\bin\Release\</OutputPath>
    <BuildScriptName>$(MSBuildProjectName).sql</BuildScriptName>
    <!-- ... -->
  </PropertyGroup>
  <ItemGroup>
    <Build Include="Tables\MyTable.sql" />
    <None Include="Publish\DB.publish.xml" />
  </ItemGroup>
  <ItemGroup>
    <ArtifactReference Include="..\..\_DACPACs\frontend.dacpac">
      <!-- ... -->
    </ArtifactReference>
  </ItemGroup>
</Project>
```
