
# DataSource name is too short

|||
|:-|:-|
| Id | **RDL0007**
| Mnemo | DATASOURCE_NAME_LENGTH
| Severity | ⚠ Warning
| Category | [Report](./Group_Report.md)
| Source code | [DatasourceReferenceNameMinLengthRule.cs](../../../Rules/Reports/DatasourceReferenceNameMinLengthRule.cs)

## Cause

This rule is triggered if the length of a DataSource's name is 1.
<p id="descr">Give the DataSource a more meaningful name.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="..">
  <DataSources>
    <DataSource Name="D">
    </DataSource>
  </DataSources>
</Report>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSources>
    <DataSource Name="MY_DS">
    </DataSource>
  </DataSources>
</Report>
```
