
# Unresolved DataSet reference

|||
|:-|:-|
| Id | **RDL0013**
| Mnemo | UNRESOLVED_DATASET_REFERENCE
| Severity | ⚠ Warning
| Category | [Report](./Group_Report.md)
| Source code | [UnresolvedDatasetReferenceRule.cs](../../../Rules/Reports/UnresolvedDatasetReferenceRule.cs)

## Cause

This rule is triggered if a report contains a reference to an unknown DataSet.
<p id="descr">Specify a DataSet or use the existing one.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS">
    </DataSet>
  </DataSets>
  <ReportSections>
    <ReportSection>
      <Body>
        <ReportItems>
          <Tablix Name="Tablix1">
            <DataSetName>UNKNOWN_DS</DataSetName>
          </Tablix>
        </ReportItems>
      </Body>
    </ReportSection>
  </ReportSections>
</Report>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS">
    </DataSet>
  </DataSets>
  <ReportSections>
    <ReportSection>
      <Body>
        <ReportItems>
          <Tablix Name="Tablix1">
            <DataSetName>MY_DS</DataSetName>
          </Tablix>
        </ReportItems>
      </Body>
    </ReportSection>
  </ReportSections>
</Report>
```
