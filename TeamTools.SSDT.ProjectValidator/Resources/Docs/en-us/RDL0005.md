
# DataSet name contains illegal symbols

|||
|:-|:-|
| Id | **RDL0005**
| Mnemo | DATASET_NAME_ALLOWED_SYMBOLS
| Severity | ⚠ Warning
| Category | [Report](./Group_Report.md)
| Source code | [DatasetNameAlphanumericRule.cs](../../../Rules/Reports/DatasetNameAlphanumericRule.cs)

## Cause

This rule is triggered if a DataSet name contains illegal symbols.
<p id="descr">Remove all illegal symbols from the DataSet name.</p>

## Examples

Bad

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="$Ё">
    </DataSet>
  </DataSets>
</Report>
```

Good

```xml
<?xml version="1.0" encoding="utf-8"?>
<Report MustUnderstand="df" xmlns="...">
  <DataSets>
    <DataSet Name="MY_DS">
    </DataSet>
  </DataSets>
</Report>
```

## Tips

💡 It is prohibited to use any symbols other than Latin letters, dashes (-), underscores (_) or periods (.).
